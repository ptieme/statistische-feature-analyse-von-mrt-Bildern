# statistische-feature-analyse-von-mrt-Bildern
Statistische Feature-Analyse von MRT-Bildern (Alzeimer's Disease Dataset). Extraktion einfacher Intensitätsmerkmale (mean, Varianz, Skewness), Visualisierung und binäre Klassifikation (Demented vs. NonDemented) mittels logistischer Regression und ROC-Auc.
