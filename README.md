# Statistische Feature-Analyse von  MRT-Bildern

## 1. Motivation und Relevanz des Thema

Die Analyse medizinischer Bilddaten spielt eine zentrale Rolle in der  modernen Medizin, insbesondere bei der Fhrüherkennung  und Diagnose neurodegenerativer Erkrankungen wie Alzeihmer. MRT-Bilder lieferm wichtige  Informationen über strukturelle Veränderungen des Gehirns, deren quantitative Analyse zur Unterstützung Klinischer Entscheidungsprozesse beitragen kann.

Ziel dieses Projekts ist es,  zu untersuchen, inwiefern einfache  statistische Merkamle aus MRT-Bildintensitäten genutzt werden können, um zwischen gesunden und dementen Patienten zu unterscheiden.

## 2. Zielsetzung des Projekts 

Das Projekt verfolgt folgende Ziele:

- Verständnis  der Struktur und Eingenschaften eines vorverarbeiteten MRT-Datensatzes.
- Extraktion einfacher statistischer Features aus Bildern.
- Explorative Datenanalyse und Visualisierung.
- Untersuchung der Aussagekraft der extrahierten Features.
- Durchführung einer binären Klassifikation (Demented vs. NonDemented).
- Kritische Reflexion der Ergebnisse  und methodischen Entscheidungen.



## 3. Projektstruktur

Das Projekt  ist wie folgt aufgebaut:

- **Jupyter Notebook:**
Enthält den vollständigen Code zur Datenanalyse, Feature-Extraktion, Visualisierung und Modellierung. Der Fokus hier  liegt auf sauberem, nachvollziehbarem und Kommentiertem Code 

- **Gitlab Wiki:**
Detaillierte Dokumentation jeder Projektphase mit Brgründungen, Beobachtungen, Problemen und Reflexionen.
